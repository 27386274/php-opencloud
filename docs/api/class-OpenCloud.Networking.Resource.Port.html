Class Port

A port represents a virtual switch port on a logical network switch, represented by OpenCloud\Networking\Resource\Network. Virtual instances (such as servers created using the OpenCloud\Compute\Service) attach their interfaces into ports. The port also defines the MAC address and the IP address(es) to be assigned to the interfaces plugged into them. When IP addresses are associated to a port, this also implies the port is associated with a OpenCloud\Networking\Resource\Subnet, as the IP address is taken from the allocation pool for a specific subnet.
