    ServerList is a Collection of servers

    Collection objects are used whenever there are multiples; for example, multiple objects in a container, or multiple servers in a service.

    author Glen Campbell

    Inherited

    This function should be called after any json_*() function call. This ensures that nasty JSON errors are detected and the proper exception thrown.

    Example: $obj = json_decode($string); if (check_json_error()) do something ...

    Inherited

    The primary parameter is a string in sprintf() format, and it can accept up to five optional parameters. It prints the debug message, prefixed with "Debug:" and the class name, to the standard output device.
