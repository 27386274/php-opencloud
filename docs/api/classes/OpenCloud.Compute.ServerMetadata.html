    This class handles server metadata

    Server metadata is a weird beast in that it has resource representations and HTTP calls to set the entire server metadata as well as individual items.

    author Glen Campbell <glen.campbell@rackspace.com>s

    check the most recent JSON operation for errors

    CheckJsonError() : boolean
    Inherited

    This function should be called after any json_*() function call. This ensures that nasty JSON errors are detected and the proper exception thrown.

    Example: $obj = json_decode($string); if (check_json_error()) do something ...
